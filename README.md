## Overview

**Pixel Pull** is a sleek Android application that automatically downloads and sets wallpapers from any custom URL on a daily schedule. Perfect for dynamic wallpapers, daily art feeds, or personalized calendar images.

##  Features

- ** Custom URL Support** - Use any publicly accessible image URL as your wallpaper source
- ** Scheduled Updates** - Set a specific time for automatic daily wallpaper refreshes
- ** Manual Updates** - Update your wallpaper instantly with a single tap
- ** Material3 Dark Theme** - Beautiful, modern dark grey interface
- ** Persistent Scheduling** - Continues working after device reboots
- ** Direct Setting** - No preview screens, wallpaper sets instantly

##  Getting Started

### Quick Start Guide

1. **Enter URL** - Input your wallpaper image URL (or use the default random image source)
2. **Set Schedule** - Tap "Schedule Time" to choose when your wallpaper updates daily
3. **Update Now** (Optional) - Tap "Update Wallpaper Now" to apply immediately
4. **Enjoy** - Your wallpaper will automatically refresh at the scheduled time every day!

### Using Custom Wallpaper Sources

Pixel Pull works with any publicly accessible image URL. Some popular options:

- **Life Calendars**: `https://lifecal-virid.vercel.app/months?height=2340&width=1080`
- **Custom APIs**: Any service that returns image URLs with your device dimensions

 **Tip**: For best results, use URLs that match your device's screen resolution.

## Installation

1. Download the APK file
2. Enable **"Install from Unknown Sources"** in Android settings
3. Open the APK and follow installation prompts
4. Launch Pixel Pull and configure your preferences
